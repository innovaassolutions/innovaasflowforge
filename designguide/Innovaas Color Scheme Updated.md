# Innovaas Website Color Scheme Guide

## 1. Core Brand Colors
| Role              | Hex Code | Notes |
|-------------------|----------|-------|
| **Primary Accent (Orange)** | `#F25C05` | Energetic, innovative, used for CTAs, highlights, and icons. |
| **Primary Text (White)** | `#FFFFFF` | Main typography on dark background. |
| **Navbar Background** | `#1E232D` | Dark navy-gray tone used for header navigation bar. |
| **Main Background** | `#111928` | Deep navy charcoal used behind most content sections. |
| **Footer Background** | `#171B26` | Slightly different dark navy charcoal used in footer strip. |

---

## 2. Secondary & Supporting Colors
| Role              | Hex Code | Notes |
|-------------------|----------|-------|
| **Secondary Accent (Teal Blue)** | `#1D9BA3` | Adds trust and balance; use for links, banners, secondary CTAs, and info highlights. |
| **Neutral Gray Light** | `#F5F5F5` | Light backgrounds, cards, and hover states. |
| **Neutral Gray Mid** | `#777777` | Secondary text, borders, and dividers. |

---

## 3. Accessibility & Contrast
- White text on all dark nav/footer backgrounds passes **AAA**.  
- Orange (`#F25C05`) on dark background passes AA for large buttons and headers.  
- Avoid orange text on **white/light** backgrounds (fails AA for small text). Use darker orange (`#D94C04`) for text labels.  
- Teal (`#1D9BA3`) on dark backgrounds passes AA.  

---

## 4. Usage Rules
1. **Primary CTAs (Buttons):**  
   - Background: Orange `#F25C05`  
   - Text: White `#FFFFFF`  
   - Hover: Darker orange `#D94C04`  

2. **Secondary CTAs / Info Buttons:**  
   - Background: Teal `#1D9BA3`  
   - Text: White `#FFFFFF`  
   - Hover: Dark teal `#14737A`  

3. **Links:**  
   - Default: Teal `#1D9BA3`  
   - Hover: Dark teal `#14737A`  

4. **Headings & Body Text:**  
   - On dark: White `#FFFFFF`  
   - On light: Main background (`#111928`)  

5. **Backgrounds:**  
   - Navbar: `#1E232D`  
   - Main body: `#111928`  
   - Footer: `#171B26`  
   - Light cards/sections: `#F5F5F5`

## 6. Summary
- Your site already uses **deep navy-charcoal backgrounds** (`#1E232D`, `#111928`, `#171B26`).  
- Keep orange as the **primary brand accent**.  
- Introduce teal as a **secondary accent** to diversify visuals and add trust signaling.  
- Ensure WCAG accessibility rules for all text/background combinations.
