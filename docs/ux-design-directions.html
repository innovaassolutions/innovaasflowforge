Design Patterns Applied
Visual Hierarchy Stats at top for quick scan, campaigns below for action items. Primary CTA (New Campaign) prominent in header.
Color Usage Orange accent used sparingly for CTAs, progress, and active states. Status badges use semantic colors.
Whitespace Generous padding (32-40px) creates breathing room. Card gaps maintain visual separation.

That's a common challenge. Can you tell me about any specific situations where this disconnect has caused problems or delays?
Design Patterns Applied
Focus Mode No sidebar, minimal header. Full attention on the conversation. Progress bar provides context without distraction.
Conversation Design Chat bubbles differentiate AI from user. Avatars humanize the interaction. Message alignment creates natural flow.
Calm Palette Soft backgrounds reduce cognitive load. Orange accent only on send button - the single action available.
